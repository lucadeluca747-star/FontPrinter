## FontPrinter
![Font Printer Screenshot](./Font-printerimg.jpeg)
Font printer is a tool used to print an input string with every font in the library of pyfiglet which is intalled trough "pip install pyfiglet" and also uses the library: "keyboard" installed trough "pip install keyboard" command in bash

### ***!NOTE!***
it is very probable that it will give an error during first use, **please don't touch it**, the code is designed to install missing libraries or fix variables value in case of an error, then the program will restart but in a secondary provvisory way, so after downloading the code and starting it for the first time, please, try to restart the ccode, and if things go bad or the program isn't able to recover from an unexpcted mistake, pls hit the **esc** key on your keyboard and if this doesn't succed restart the program, thank you for taking your time to read the README.md file, have a nice day!
